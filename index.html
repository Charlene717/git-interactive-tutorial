From your first commit to full GitHub collaboration — 12 modules covering core Git, branching strategies, GitHub workflows, and reproducible bioinformatics pipelines.

Step 1

Step 4

Step 6

Step 7

remote / push / pull / fetch, tracking branches, and authentication via SSH or PAT.
🤝
Step 8

GitHub 協作 (PR & Review)

fork、Pull Request、code review、Issue、Draft PR、Projects 看板與權限。

fork, pull requests, code review, issues, draft PRs, project boards, and permissions.

Step 9

.gitignore & Git LFS（生物資訊）
